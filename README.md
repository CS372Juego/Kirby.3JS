# Kirby.3JS
- Jason Chew, Daniel Kim

## How to Play
- WASD
- Space: Jump
- Shift + WASD: Sprint
- UpArrow: Teleport!
    - Teleport at the end of the world
- Click [here](https://students.cs.calvin.edu/~jk254/372/proj6/) to play!
